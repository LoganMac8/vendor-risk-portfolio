# üîÑ Control Alignment Summary  
Across all three frameworks, several universal themes emerged:

### **Common Areas of Strong Alignment**
- Access control & least privilege  
- Encryption of data in transit and at rest  
- Logging, monitoring, and auditability  
- Secure development practices  
- Vendor and subprocessor management  
- Incident response planning  
- Business continuity expectations  

These control families map consistently, making them strong areas of overlap.

# ‚ö†Ô∏è Key Differences Noted
Some frameworks are more prescriptive than others:

### **NIST 800-53**
- Most detailed and technical  
- Requires explicit documentation and evidence trails  
- Includes rigorous requirements for baseline configuration, rollback plans, and periodic testing  

### **ISO 27001**
- Balanced and process-driven  
- Strong emphasis on supplier lifecycle and governance  
- Allows flexibility as long as controls are documented and justified  

### **SOC 2**
- Principles-based  
- Less prescriptive, more outcome-focused  
- Depends on service auditor interpretation  

Because of these differences, organizations may meet SOC 2 but fall short of ISO or NIST without additional rigor.

# üß© Gap Insights  
Several gaps were identified when aligning all three frameworks:

- Limited rollback documentation in change management  
- Inconsistent access review scheduling  
- Lack of detailed data lifecycle documentation  
- Cloud configuration checks not always automated  
- Incident response exercises not performed annually  

Each gap includes a clear remediation path in the accompanying `Gaps-and-Recommendations.md` file.

# üìà Maturity Observation  
A security program aligned to NIST-level rigor generally satisfies SOC 2 and ISO requirements as well.  
The strongest programs:

- Maintain detailed documentation  
- Test controls regularly  
- Automate evidence collection  
- Apply governance consistently across teams  
- Treat compliance as an active, ongoing process  

This mapping exercise demonstrates those same principles.
